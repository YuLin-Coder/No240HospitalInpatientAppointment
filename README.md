## 项目介绍
基于springboot的医院住院预约管理系统：前端 html、jquery、layui，后端 maven、springmvc、spring、mybatis；角色分为管理员、医生、患者；集成医生管理、患者管理、药品管理、预约管理、病史管理等功能于一体的系统。
